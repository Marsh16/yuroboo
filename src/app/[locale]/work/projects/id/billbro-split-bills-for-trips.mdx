## Ringkasan

BillBro for Macro Challenge di Apple Developer Academy @ UC, merupakan aplikasi yang dapat membantu sekelompok orang yang sedang melakukan perjalanan liburan bersama untuk membagi tagihan atas kegiatan mereka secara lebih detail, sehingga beberapa tagihan dapat dibagi dalam 1 aplikasi dan tidak terpecah-pecah, serta mereka bisa mendapatkan invoice mereka dengan lebih lengkap dan perhitungan yang lebih terpercaya dengan fitur input manual atau scan nota dengan memanfaatkan OCR (Optical character recognition). OCR adalah sebuah sistem yang digunakan untuk mengubah teks atau gambar menjadi teks digital. Fitur-fitur yang ada di dalam aplikasi ini dapat membantu pengguna lebih mudah membagi tagihan mereka saat bepergian berlibur bersama. 

## Fitur Utama

- **Scan dan Konversi Bill**: Scan tagihan Anda dan buat detailnya secara otomatis.
- **Bagi Pengeluaran Secara Adil**: Tetapkan Tagihan dengan perhitungan pembagian yang adil.
- **Lacak Semua Pengeluaran Anda**: Simpan semua tagihan Anda di satu tempat dan lihat siapa yang berhutang kepada Anda.
- **Bagikan Ringkasan Hutang**: Kirim rincian pengeluaran secara instan ke grup Anda.

## Teknologi yang Digunakan

- **OCR Scan**: Menggunakan OCR untuk Input bill dengan mengambil gambar/memilih gambar struk. Menggunakan Gemini AI API dari Pihak Ketiga
- **Widget**: Akses Lebih Cepat untuk Memasukkan Tagihan. Menggunakan SwiftUI dan SwiftData
- **Notifikasi**: Pembuat Perjalanan mendapatkan Notifikasi setelah perjalanan berakhir H+1 dan H+3 untuk mengingatkan peserta mereka. OneSignal (layanan untuk mengirim pemberitahuan push) Pihak Ketiga, Supabase untuk database online, sehingga dapat diintegrasikan dengan onesignal untuk mengirim pemberitahuan push.
- **Perjalanan Aktif**: BillBro memiliki perjalanan aktif sehingga pengguna dapat memilih perjalanan mereka lebih cepat di widget. Menggunakan SwiftUI dan Swift.
- **Bagikan PDF**: Bagikan perjalanan dalam File Faktur PDF ke peserta Anda atau simpan file pdf. Menggunakan SwiftUI dan PDFKit.
- **Salin Nomor Rekening**: Salin nomor dan transfer. Menggunakan SwiftUI.
- **Database**: SwiftData dan Supabase (hanya untuk notifikasi)

## Tech Stack

- **Teknologi Apple**: SwiftUI, SwiftData, WidgetKit, Notifications, PDFKit
- **Pihak Ketiga**: OneSignal, Gemini API, Supabase

## Hasil

BillBro berhasil mengatasi tantangan membagi pengeluaran di antara teman selama perjalanan kelompok. Dengan memanfaatkan teknologi canggih seperti OCR, API Gemini yang didukung AI, dan layanan berbasis cloud, aplikasi ini menawarkan solusi yang ramah pengguna dan efisien.

Proyek ini menunjukkan penerapan teknologi mutakhir yang berhasil untuk memecahkan masalah dunia nyata, menunjukkan potensi solusi inovatif untuk meningkatkan pengalaman perjalanan dan meningkatkan interaksi sosial.
