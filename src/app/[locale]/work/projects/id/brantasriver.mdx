## Ringkasan

Sungai Brantas merupakan game mobile endless runner yang mengambil lokasi di sungai brantas, sungai terpanjang di Provinsi Jawa Timur.
Dalam permainan tersebut, pemain berperan sebagai perahu yang menyusuri sungai sambil menghindari bebatuan yang berserakan di sepanjang jalan. Saat mereka bepergian, mereka mengambil sampah sebanyak yang mereka bisa.
Pemain perlu memiringkan ponselnya ke kiri atau kanan untuk mengontrol pergerakan perahu. Babi hutan akan tenggelam jika menabrak batu dan permainan berakhir.

## Fitur Utama

- **Tilt to Play**: Memiliki fitur gerakan gerak, untuk menggerakkan pemutar dan mengumpulkan item, miringkan ponsel Anda
- **Collect Trash and Avoid Rocks**: Dapat mengumpulkan sampah dan mati jika terbentur batu

## Teknologi yang Digunakan

- **Motion Sensors**: Menggunakan CoreMotion untuk sensor gerak dalam game
- **3D Object**: Menggunakan SceneKit untuk membuat lingkungan 3D dan blender untuk membuat Objek 3D.

## Tech Stack

- **Apple Techs**: GameplayKit, Core Motion, SceneKit, SwiftUI, Core Haptics, Audio, Blender 

## Tantangan dan Pembelajaran

Game iOS yang seru untuk dimainkan orang dan mengumpulkan sampah di sungai brantas menggunakan teknologi apple seperti GameplayKit, Core Motion, SceneKit, SwiftUI

## Hasil

SceneKit, Core Motion dan mengintegrasikan GameplayKit dalam game 3d untuk Arsitektur ECS, Audio dalam game, dan Core Haptics
