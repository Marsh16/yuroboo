## Gambaran Umum

Merokok dan minum adalah masalah kesehatan utama secara global, yang menyebabkan berbagai penyakit seperti kanker, penyakit paru-paru, dan penyakit hati. Identifikasi dini kebiasaan ini sangat penting untuk tindakan pencegahan. Aplikasi HealthSignal dirancang untuk mengatasi tantangan ini dengan memanfaatkan machine learning untuk memprediksi kebiasaan merokok dan minum berdasarkan data sinyal tubuh yang dikumpulkan dari pengguna.

## Cara Kerja

HealthSignal adalah aplikasi seluler Android yang menggunakan model machine learning untuk menganalisis data sinyal tubuh dan memprediksi kemungkinan pengguna menjadi perokok atau peminum. Aplikasi ini menggunakan dua model machine learning utama:

- **Klasifikasi Random Forest**: Model ini sangat efektif untuk mengidentifikasi peminum berdasarkan pola sinyal tubuh.
- **Klasifikasi Multi-Layer Perceptron (MLP)**: Model ini, juga dikenal sebagai Klasifikasi Jaringan Neural.

Kedua model tersebut dioptimalkan menggunakan penyetelan hyperparameter Grid-Search atau Random-Search untuk memastikan akurasi maksimum.

## Manfaat

HealthSignal menawarkan beberapa manfaat bagi pengguna dan profesional perawatan kesehatan:

- **Deteksi Dini**: Aplikasi ini berpotensi membantu dalam identifikasi dini kebiasaan merokok dan minum, memungkinkan intervensi tepat waktu dan tindakan pencegahan.
- **Peningkatan Manajemen Kesehatan**: Dengan memprediksi kebiasaan ini, penyedia layanan kesehatan dapat memperoleh wawasan berharga tentang risiko kesehatan pasien, memungkinkan mereka untuk mengembangkan rencana perawatan yang lebih personal.
- **Kesadaran Pengguna**: Aplikasi ini dapat memberdayakan pengguna untuk mendapatkan wawasan tentang kesehatan mereka dan membuat keputusan yang tepat terkait kebiasaan merokok dan minum.

## Tumpukan Teknologi

Aplikasi HealthSignal dibangun menggunakan teknologi berikut:

- **Android Studio**: Platform pengembangan utama untuk aplikasi seluler.
- **TensorFlow Lite (TFlite)**: Kerangka machine learning yang memungkinkan penerapan model machine learning pada perangkat Android.
- **Model Machine Learning**: Klasifikasi Random Forest dan Multi-Layer Perceptron digunakan untuk prediksi.
- **Python**: Menggunakan Python untuk bahasa pemrograman

## Hasil

Aplikasi HealthSignal menyajikan pendekatan yang menjanjikan untuk memanfaatkan machine learning untuk memprediksi kebiasaan merokok dan minum berdasarkan data sinyal tubuh. Dengan menawarkan deteksi dini dan meningkatkan kesadaran pengguna, HealthSignal berpotensi meningkatkan tindakan pencegahan kesehatan dan memberdayakan individu untuk membuat pilihan yang tepat terkait kesehatan mereka.
